Query with Terms

Instead of using the built-in correlation searches, users can construct and run their own queries by selecting terms from the term list.

The LSA component treats the user's list as a mini-document and searches for chunks that correlate with it. A user query with terms will return correlated chunks and report which terms are present in those chunks.

This is similar to the Term-Chunk correlatiion search. See Term-Chunk correlations

Because the query string will be much shorter than 250 or 1000 words, use a lower correlation value when running the search.
