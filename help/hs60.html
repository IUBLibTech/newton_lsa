Chunk-Chunk correlations

The Chunk-Chunk search is an extension of the Document-Document search type.  Document-Document correlations

In the Document-Document search, you select a document and the LSA component returns results for all the chunks in that document.

In the Chunk-Chunk search, you can select particular chunks of one or more documents and the LSA component will only return results for the selected chunks and not for whole document or documents.
