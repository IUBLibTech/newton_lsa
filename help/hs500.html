Step 6 -- Work with the Results List

When a search succeeds the LSA component will return a list of results like the following list for a Document-Document search.

Click one of the correlation links to view the chunk pair side by side.

Significant shared vocabulary will be highlghted in yellow for Document-Document and Chunk-Chunk searches.

In Term-Chunk and Query by term searches, the selected terms will be highlighted in blue.

At the top of each chunk, a link will be provided that will take you to page in the digital edition which contains that text.

Examination will frequently show that the pairs contain shared text even when that text is not word-for-word identical.
LSA can be a very robust search tool.
