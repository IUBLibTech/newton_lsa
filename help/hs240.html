All Between Docs or Terms

This scope option only returns result pairs if both members (i.e. both chunks or both terms) are part of the query set selected by the user.

This allows users to view only the connections between the particular documents or terms they're interested in.

This option excludes results from all chunks and terms not in the query set, even if their correlation values are greater than the chosen correlation level.

This option is only available for Document-document searches and Term-term searches. It is not available for Chunk-chunk searches.
